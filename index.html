ชื่อวิชาComputer Network and Internetเครือข่ายคอมพิวเตอร์และอินเทอร์เน็ต
ภาพรวมของรายวิชานี้จะพาผู้เรียนจากพื้นฐาน ไปสู่การทำความเข้าใจการทำงานของอินเทอร์เน็ตและเครือข่ายคอมพิวเตอร์
  • ศึกษาการสื่อสารข้อมูลและองค์ประกอบของระบบเครือข่าย
  • การส่งข้อมูล ตัวกลาง และการควบคุมการเชื่อมโยงข้อมูล
  • เครือข่ายเฉพาะที่ (LAN) และเครือข่ายบริเวณกว้าง (WAN)
  • สถาปัตยกรรมการสื่อสารและโพรโทคอล TCP/IP
  • การออกแบบเลขที่อยู่ IP และโพรโทคอลการจัดเส้นทาง
  • การเชื่อมโยงระหว่างเครือข่ายและการประมวลผลแบบ Client–Server

Why this course matters

1

ใช้งานทุกวัน

การเปิดเว็บไซต์ ส่งข้อความ ประชุมออนไลน์ และใช้ Cloud ล้วนเกี่ยวข้องกับเครือข่าย

2

เข้าใจระบบจริง

ผู้เรียนจะเห็นว่าข้อมูลเดินทางผ่านอุปกรณ์ ตัวกลาง โพรโทคอล และเส้นทางอย่างไร

3

ต่อยอดงาน IT

เป็นพื้นฐานสำหรับ Web, Server, Cybersecurity, IoT, Cloud และระบบองค์กร

คำถามนำเข้าสู่บทเรียน: เมื่อเราเปิดเว็บไซต์หนึ่งครั้ง ข้อมูลเดินทางผ่านอะไรบ้าง?
